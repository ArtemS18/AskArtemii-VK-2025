## Описание
Сайт является аналогом StackOverflow ( задавать вопросы, отвечать на них).
Присутствует пользовательская рейтинговая система, полноценная сессионная авторизация, возможность загрузки изображений на удаленный сервер Minio или в корень проекта (/media). 

## Описание структуры репозитория

- `main` — главная ветка (25.12.2025 merge с hw7) 
- `hw1` — Добавлены базовые шаблоны
- `hw2` — Добавлен роутинг по шаблонам
- `hw3` — Добавлен Postgres DB.
- `hw4` — Добавлена авторизации через сесси. Добавлен Redis для их хранения.
- `hw5` — Добавлены AJAX запросы для оценок вопросов и комментарием. Добавлено S3 Minio для изображений.
- `hw6` — Настроен nginx в качестве основного прокси-сервера, настроен proxy_cache.
          Проведены сранительные тесты производительности.
- `hw7` — Добавлены real-time сообщения по centrifugo.
          Добавлен комбинированный полнотекстовый поиск с использованием ts_vector.
          Добавлен сron-скрипт для проведение расчетов.
          Добавлено кеширование запросов к БД и инвалидация по ключу.

## Запуск 
`docker compose up -d`
`poetry install`
`poetry run main`
